# JetBrains Kotlin Blog

## 網站簡介

這是 JetBrains 的官方 Kotlin Blog，也是我覺得很值得長期追的一個入口。

它的重點不只放在 Android，而是會從整個 Kotlin 生態系的角度去更新內容，例如語言版本釋出、Compose Multiplatform、Kotlin Multiplatform、生態工具、IDE 支援，以及和其他框架或服務的整合。

如果你不想只把 Kotlin 當成 Android 語言來看，這個 Blog 很重要。

## 我為什麼推薦

- 這裡最適合追 Kotlin 官方最新動向。
- 發版資訊通常很快，適合拿來關注升級重點與新特性。
- 不只 Android，Multiplatform、Ecosystem、工具鏈也都常有內容。
- 很多文章會帶到實際技術方向，例如 Compose Multiplatform、Spring、AI tooling 或 IDE 整合。

## 這個 Blog 特別適合補什麼

- Kotlin 新版本與 release notes
- Kotlin Multiplatform 與 Compose Multiplatform 動態
- Kotlin 工具鏈與 IDE 功能進展
- Kotlin 生態系的新工具、新套件與實作文章
- 想知道 Kotlin 現在整體在往哪裡走

## 我會怎麼看這個 Blog

- 要升級 Kotlin 版本之前，先看 release 文章。
- 想追多平台方向時，看 `Multiplatform` 類別。
- 想掌握整體工具與生態系更新時，看 `Ecosystem` 和最新文章列表。

## 我的簡短評價

如果 Android Developers 比較像是「Android 情境下的官方教學入口」，那 JetBrains Kotlin Blog 比較像是「整個 Kotlin 世界現在在發生什麼事」的官方窗口。

兩個都值得看，但用途不太一樣。

## 官方來源

- 官方 Blog: [Kotlin | The Kotlin Blog](https://blog.jetbrains.com/kotlin/)
- Kotlin 官網: [kotlinlang.org](https://kotlinlang.org/)

## 來源說明

JetBrains Kotlin Blog 首頁明確把內容分成 `News`、`Releases`、`Multiplatform` 與 `Ecosystem` 等主題，並持續發布 Kotlin 版本更新、Compose Multiplatform、多平台與生態工具相關文章，很適合作為追蹤 Kotlin 發展方向的官方資訊來源。
