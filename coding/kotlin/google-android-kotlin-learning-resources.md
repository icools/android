# Google 官方 Android Kotlin 學習資源

## 網站簡介

Android Developers 的 Kotlin 專區，是我會優先收藏的官方入口之一。

它不是只有一篇「Kotlin 教學」而已，而是把 Android 團隊希望你優先理解的 Kotlin 資源整理在同一個地方，包含從零開始的 Kotlin 學習、Android 常見 Kotlin 模式、樣式指南、Java 互通性、coroutines、Flow，以及 Kotlin Multiplatform 相關內容。

如果你是在 Android 情境下學 Kotlin，這一頁很適合當成總索引。

## 我為什麼推薦

- 它是 Android 官方整理的 Kotlin 入口，方向很清楚。
- 不只講語法，還會把 Android 專案裡常遇到的主題一起收進來。
- 從初學、Java 轉 Kotlin，到進階的 coroutine / Flow / KMP，都有對應資源。
- 如果你想知道「Android 官方現在比較推薦怎麼寫」，這一頁通常是很好的起點。

## 我特別想強調的一點

如果是語意比較細、容易因翻譯而產生誤解的主題，我會建議優先看英文原文。

中文版本不是不能看，但有些技術名詞、語氣上的細節，英文通常會比較精準。尤其當你在看語言設計、最佳實務、行為差異或 API 用法時，直接讀英文原文比較不容易被翻譯帶偏。

## 這一頁特別適合查什麼

- 從頭開始學 Android 與 Kotlin
- Android 中常見的 Kotlin 寫法
- Kotlin 樣式指南
- 將 Kotlin 新增到現有 Android 專案
- Kotlin 與 Java 的互通性
- coroutines、Flow、StateFlow、SharedFlow
- Kotlin Multiplatform 在 Android 這邊的官方入口

## 我會怎麼使用這份資源

- 想確認 Android 官方推薦做法時，先從這頁進去找對應主題。
- 遇到 Kotlin / Java 互通性、coroutines 或 Flow 的細節時，優先回官方文件確認。
- 如果是要帶團隊建立共識，樣式指南和官方教學特別值得先看。
