# Kotlin Playground

## 平台簡介

`Kotlin Playground` 是一個可以直接在瀏覽器裡撰寫、執行與分享 Kotlin 程式碼的官方線上平台。

如果你只是想快速驗證某段 Kotlin 語法、試一個小範例，或把一段可執行的 code 直接丟給別人看，這個平台非常方便，因為你不需要先開新專案，也不需要先把環境都架好。

## 我為什麼推薦

- 很適合快速測試 Kotlin 語法和小片段邏輯。
- 不需要先建立專案，打開瀏覽器就能跑。
- 可以切換不同 Kotlin 版本與執行環境。
- 測完之後可以直接分享連結，教學或討論時很好用。

## 這個平台可以做什麼

- 在瀏覽器中直接編寫與執行 Kotlin 程式
- 測試小型函式、標準函式庫用法與語法片段
- 切換不同執行後端，例如 JVM、JS 或 JUnit
- 在 JS 後端下查看產生的 JavaScript
- 直接複製分享連結，把可執行範例丟給別人

## 很適合拿來做哪些事

- 驗證一段語法是不是你理解的那樣
- 快速測試 collection、scope functions、nullable、lambda 等小範例
- 教學時提供可直接執行的程式片段
- 討論問題時，附上一個可重現的小案例

## 我會怎麼使用它

- 先在 Playground 驗證想法，再回專案裡正式實作。
- 想和別人討論一段 Kotlin 寫法時，直接分享 Playground 連結。
- 遇到只需要小型重現案例的問題時，用它比開新專案快很多。

## 官方來源

- Playground: [Kotlin Playground](https://play.kotlinlang.org/)
- 官方說明: [Run code snippets](https://kotlinlang.org/docs/run-code-snippets.html)

## 來源說明

Kotlin 官方文件把 Kotlin Playground 定義成可在瀏覽器中撰寫、執行與分享 Kotlin 程式碼的線上工具，並說明它支援多個 Kotlin 版本、多種執行後端，例如 JVM、JS 與 JUnit；在 JS 模式下也可以查看產生的 JavaScript，並且能直接複製分享連結或嵌入程式片段。
