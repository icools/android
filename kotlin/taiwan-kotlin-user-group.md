# 台灣 Kotlin User Group

## 網站簡介

`Taiwan Kotlin User Group` 是台灣的 Kotlin 社群網站，首頁就直接寫明它是在台灣推廣 Kotlin、舉辦相關活動，歡迎對 Kotlin 有興趣的人一起交流與聚會。

我覺得它很有價值的地方，不只是有社群活動，而是站上本身就整理了不少內容，從新人專區、推薦文章，到進階範例、套件介紹、合作社群與活動歷程，都很完整。

## 我為什麼推薦

- 它是很在地的 Kotlin 社群入口，閱讀和參與門檻都比較低。
- 除了活動資訊，站上本身就有很多可直接閱讀的技術文章。
- 新人、前端、後端、函數式程式設計等不同背景的人，都有對應入口。
- 文章主題不只停在入門，也涵蓋 data class、sealed class、delegation、KDoc、Ktor、Koin 等延伸內容。

## 這個網站適合怎麼看

如果你剛接觸 Kotlin，可以先從首頁的新人專區開始，看語法特色、慣用寫法與入門範例。

如果你已經有一些基礎，可以往進階範例、Ktor / Koin 這類套件介紹，或是合作夥伴與推薦文章繼續延伸。它很適合當成「官方文件之外，補台灣社群觀點」的入口。

## 社群活動也很值得追

首頁有完整的活動歷程，可以看到近年的 Kotlin Meetup、KotlinConf watch party 和不同場地舉辦的實體活動。

如果你想找能交流、能聽分享、也能認識在地開發者的社群，這個網站很值得收藏。

## 我會怎麼使用這份資源

- 想找中文脈絡的 Kotlin 文章時，先來這邊挖。
- 想補實作範例時，先看它整理好的入門與進階主題。
- 想知道台灣最近有哪些 Kotlin 社群活動，也可以直接看首頁活動歷程。
