# Decompile Tools

這裡整理反編譯與分析 APK 結構時常用的工具。

- [`jadx-ui`](jadx-ui.md) - 很適合拿來查看 APK、Dex、resources 與反編譯結果，是我理解陌生 App 結構時最常想到的第一站。
- [`LibChecker`](libchecker.md) - 直接在手機上檢查已安裝 App 使用哪些函式庫，適合在真正打開反編譯工具前先做第一輪技術偵察。
